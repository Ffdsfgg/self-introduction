## 🎯 项目详细介绍

### [AI智能教学视频系统](https://github.com/Ffdsfgg/video-ai-agent-master)

**项目定位**: 融合人工智能与在线教育的智能视频教学平台

#### 核心功能
- **智能视频处理**: 自动生成字幕、知识点标注
- **内容检索**: 基于AI的精准内容搜索
- **学习分析**: 学习行为跟踪与个性化推荐
- **教学管理**: 完整的后台管理系统

#### 技术架构
- **后端**: Spring Boot 3 + MySQL + MyBatis-Plus
- **前端**: Vue 3 + Element Plus
- **AI服务**: 语音识别、自然语言处理
- **存储**: 阿里云OSS + 本地存储

#### 项目特色
> "重新定义在线教育，让每个学习者都能获得个性化的教学体验"

---

### [居无忧智能生活助理](https://github.com/Ffdsfgg/deaf-mute)

**项目定位**: 面向聋哑人群体的智能生活辅助系统

#### 核心功能
- **环境声识别**: 识别门铃、警报等关键声音
- **语音转文字**: 实时语音转换辅助沟通
- **文字转语音**: 双向交流桥梁
- **手语识别**: 手语动作识别与转换

#### 技术架构
- **硬件层**: ESP32-S3 + 鲲鹏派
- **AI层**: TensorFlow Lite + OpenCV + MediaPipe
- **服务层**: Spring Boot + WebSocket
- **移动端**: UniApp多端适配

#### 项目特色
> "不是替代耳朵，而是延伸感知 - 我们不做'治愈'，只做'赋能'"

---

### [两岸e家](https://github.com/Ffdsfgg/second-hand-market-back)

**项目定位**: 连接福建与台湾的跨境服务平台，促进两岸交流与商业合作

#### 核心功能
- **跨境服务**: 两岸商品交易、服务对接
- **政策咨询**: 两岸政策解读、投资咨询
- **商务对接**: 企业合作、项目对接服务
- **信息平台**: 两岸资讯、文化交流
- **一站式服务**: 为两岸用户提供便捷的跨境服务体验

#### 技术架构
- **后端**: Spring Boot + MyBatis-Plus
- **前端**: Vue 3 + Element Plus
- **数据库**: MySQL + Redis
- **部署**: 云服务器部署

#### 项目特色
> "连接闽台，服务两岸 - 搭建两岸交流的数字桥梁"

### [博客微服务系统](https://github.com/Ffdsfgg/blog-sql-cloud)

**项目定位**: 技术练习与微服务架构探索平台

#### 核心功能
- **博客管理**: 文章发布、编辑、分类
- **用户系统**: 多角色权限管理
- **内容搜索**: Lucene全文检索
- **社交功能**: 关注、评论、收藏
- **积分体系**: 用户等级与积分系统

#### 技术架构
- **微服务**: SpringCloud Alibaba全家桶
- **数据库**: MySQL + MongoDB + Redis
- **搜索**: Elasticsearch + Lucene
- **消息**: RabbitMQ + 延迟队列
- **部署**: Docker容器化部署

#### 项目特色
> "统一账号体系，支持多项目单点登录 - 技术深度探索与实践"
